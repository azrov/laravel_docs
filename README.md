# Laravel 12 Türkçe Dokümantasyonu 🇹🇷

Bu proje, Laravel 12'nin resmi İngilizce dokümantasyonunun (https://laravel.com/docs/12.x) Türkçe'ye çevrilmiş halidir. Amaç, Türkçe konuşan geliştiricilerin Laravel'i en doğru ve anlaşılır kaynaktan öğrenmelerini sağlamaktır.

Çeviri, anlam bütünlüğünü koruyarak, teknik terimlere sadık kalınarak ve anlaşılır bir dil kullanılarak yapılmıştır. Kod blokları orijinal haliyle bırakılmış, açıklamalar ve başlıklar Türkçeleştirilmiştir.

## 📚 İçindekiler (Mevcut Bölümler)

Dokümantasyon aşağıdaki ana başlıklar altında toplanmıştır. Bu klasör yapısı, orijinal Laravel dokümantasyonunun mantıksal akışını takip eder.

### 01_Getting Started (Başlarken)
### 02_Architecture Concepts (Mimari Kavramlar)
### 03_The Basics (Temel Bilgiler)
### 04_Digging Deeper (Derinlemesine İnceleme)
### 05_Security (Güvenlik)
### 06_Database (Veritabanı)


## 🚀 Nasıl Kullanılır?

1.  Bu depoyu (repository) bilgisayarınıza klonlayın:
    ```bash
    git clone https://github.com/[kullanici-adiniz]/laravel-12-docs-tr.git
    ```
2.  İlgilendiğiniz konunun dosyasını herhangi bir metin düzenleyicisi veya Markdown okuyucusu ile açarak okuyabilirsiniz.
3.  Dosyalar, orijinal Laravel dokümantasyonunun konu başlıklarına göre klasörlere ayrılmıştır.

## 🤝 Nasıl Katkıda Bulunulur?

Bu proje, topluluk katkılarına açıktır. Katkıda bulunmak için:

1.  Bu depoyu (repository) fork edin.
2.  Yaptığınız değişiklikleri içeren yeni bir dal (branch) oluşturun (`git checkout -b ekleme/yonlendirme`).
3.  Değişikliklerinizi commit edin (`git commit -am 'Yönlendirme bölümü eklendi'`).
4.  Dalınızı (branch) push edin (`git push origin ekleme/yonlendirme`).
5.  Bir Pull Request oluşturun.

Çeviri yaparken lütfen aşağıdaki noktalara dikkat edin:
*   Teknik terimlerin orijinal anlamlarını koruyun (örneğin, "queue" -> "kuyruk", "middleware" -> "ara yazılım").
*   Cümle yapılarını Türkçe'ye uygun hale getirin, ancak anlam kaymasına neden olmayın.
*   Kod bloklarına dokunmayın, sadece açıklama metinlerini çevirin.
*   Dosya adlarını ve bağlantıları (link) değiştirmeyin.

Detaylı bilgi için [Katkı Rehberi](contributions.md) dosyasına göz atabilirsiniz.
